# mip-caidashi-mob

mip-caidashi-mob 检测是否是移动设备访问，如果是移动设备访问，进入移动端，

标题|内容
----|----
类型|通用
支持布局|
所需脚本|https://mipcache.bdstatic.com/extensions/platform/v1/mip-caidashi-mob/mip-caidashi-mob.js

## 示例

### 基本用法
```html
<mip-caidashi-mob src="">
   
</mip-caidashi-mob>
```

### src
