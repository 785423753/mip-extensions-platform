# mip-zixun

mip-zixun 根据关键词加载商务通咨询邀请框

标题|内容
----|----
类型|通用
支持布局|responsive,fixed-height,fill,container,fixed
所需脚本|http://mipcache.bdstatic.com/static/v1/mip-zixun/mip-zixun.js

## 示例
根据关键词加载商务通咨询邀请框
### 基本用法
```html
<mip-zixun kwd="呵呵,嘿嘿"></mip-zixun>
```

## 属性

### kwd

说明：设置显示邀请框的关键词组
必选项：是
类型：字符串
